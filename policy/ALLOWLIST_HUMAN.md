# AIR Allowlist v1.0
# 작성: 2026-02-21 | 상태: ACTIVE

## 목적

AIR 에이전트가 승인 없이 할 수 있는 것과 반드시 승인이 필요한 것을 정의한다.

## 승인 없이 가능

- 리서치, 요약, 번역, 문서 초안
- 명령어/스크립트 제안 (NOT EXECUTED 표기 필수)
- GitHub 파일 읽기
- 문서 생성 및 저장 (로컬/레포)

## 승인 필수

- 토큰/키/자격증명 처리
- 시스템 설치/제거
- 권한/계정/SSH/방화벽/네트워크 변경
- LaunchAgent/OpenClaw 런타임 수정
- 모든 비가역 작업

## 절대 금지

- 시크릿/토큰 평문 저장 또는 출력
- 근거 없는 사실/수치 단정
- 승인 없이 비가역 작업 실행
