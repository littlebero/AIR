# AIR Allowlist & 승인 정책

Date: 2026-02-18
Status: ACTIVE

## 승인 없이 허용되는 작업

- 리서치, 요약, 번역, 문서 초안 작성
- 명령어/스크립트 제안 (NOT EXECUTED 라벨 필수)
- GitHub 파일 읽기
- 문서 생성 및 저장

## 반드시 승인이 필요한 작업

- credential/token/키 처리
- 시스템 설치 또는 제거
- 권한, 계정, SSH, 방화벽, 네트워크 설정 변경
- LaunchAgent 또는 OpenClaw 런타임 동작 수정
- 되돌릴 수 없는 모든 작업

## 절대 금지

- 시크릿/토큰 평문 저장 또는 출력
- 불확실한 사실/수치 단정
- 승인 없는 비가역 작업 실행
